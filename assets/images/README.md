# 이미지 넣는 방법

이 폴더(`assets/images/`)에 아래 **정확한 파일명**으로 사진을 넣으면
포트폴리오에 자동으로 표시됩니다. 파일이 없으면 기본 그래픽이 그대로 보이므로
준비된 것부터 하나씩 넣어도 됩니다. (코드 수정 불필요)

## 포트폴리오 사진 (파일명 고정)

| 파일명 | 들어가는 위치 |
|---|---|
| `portfolio-1.jpg` | 생활용품 사출 양산 |
| `portfolio-2.jpg` | 기존 제품 리브랜딩 |
| `portfolio-3.jpg` | 제작–유통 통합 |
| `portfolio-4.jpg` | 아이디어 제품화 컨설팅 |
| `portfolio-5.jpg` | 산업용 부품 양산 |
| `portfolio-6.jpg` | 타 분야 연계 제작 |

## 권장 사양

- **비율**: 16:9 (가로가 더 긴 직사각형)
- **크기**: 1200 × 675 px 정도면 충분 (너무 크면 로딩이 느려짐)
- **용량**: 한 장당 300KB 이하 권장 (가능하면 WebP, 안 되면 JPG)
- **촬영 팁**: 자연광에서 가로로, 배경 깔끔하게. 제품 단독컷 + 공정컷 조합이 좋음
- 사람이 나오면 초상권, 타사 로고가 보이면 사용 동의를 확인하세요

## 용량 줄이는 무료 도구

- https://squoosh.app (브라우저에서 바로 압축, WebP 변환)
- https://tinypng.com (JPG/PNG 압축)

## 공장 / 회사 대표 사진을 메인·소개 페이지에도 넣고 싶다면

히어로 배경이나 비전 페이지에 공장 사진을 넣는 것도 가능합니다.
사진이 준비되면 클로드에게 "공장 사진을 메인 히어로에 넣어줘"라고 요청하세요.
(예: `factory-hero.jpg` 같은 슬롯을 추가로 만들어 드립니다.)

# 문의 폼 연결하기 (중요)

문의 폼이 실제로 메일로 도착하게 하려면 **Web3Forms 키 1개**만 넣으면 됩니다.
무료이고, 가입 없이 이메일만 있으면 됩니다. (한 번만 설정하면 끝)

## 3단계

1. https://web3forms.com 접속 → 입력칸에 **jdco2209@gmail.com** 입력 후 "Create Access Key"
2. 위 메일함으로 온 **Access Key**(영문+숫자 긴 문자열)를 복사
3. `contact.html` 파일을 열어 아래 줄의 `YOUR_WEB3FORMS_ACCESS_KEY`를 복사한 키로 교체

   ```html
   <input type="hidden" name="access_key" value="YOUR_WEB3FORMS_ACCESS_KEY" />
   ```

   예) `value="a1b2c3d4-...실제키..."`
